# ChatBot as a Service 🧠

Plateforme FastAPI + Angular permettant à chaque utilisateur de créer un chatbot intelligent lié à ses propres documents PDF/txt grâce au RAG (Retrieval-Augmented Generation).

## 🔧 Technologies
- Backend: FastAPI, MongoDB, FAISS, Ollama, Sentence-Transformers
- Frontend: Angular (à venir)
- Auth: JWT
- Docker: MongoDB + Ollama + API

docker-compose up --build
